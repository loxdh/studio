# **App Name**: United Love Luxe

## Core Features:

- Product Catalog: Display a curated catalog of luxury stationery products, allowing users to browse and filter by category.
- Product Detail Pages: Showcase individual products with detailed descriptions, high-quality images, and pricing information.
- Shopping Cart Management: Enable users to add products to a shopping cart, update quantities, and proceed to checkout.
- Admin Dashboard: Provide a protected admin area for managing products, categories, and blog content.
- AI-Powered Blog Content Generation: Use the Google Gemini API as a tool to generate editorial-style articles for the blog based on user-provided titles, enhancing content creation efficiency.
- Firebase Integration: Utilize Firebase (Firestore and Storage) for managing product data and media assets.
- Theme Provider: Allow users to toggle between light and dark themes for personalized viewing preferences.

## Style Guidelines:

- Primary color: Classic gold (#D4AF37) to evoke luxury and elegance.
- Background color: Light beige (#F9F1D8) to provide a soft, warm backdrop.
- Accent color: Charcoal (#1A1A1A) to offer a sophisticated contrast.
- Headings font: 'Cormorant Garamond' (serif) for an elegant, classic feel. Note: currently only Google Fonts are supported.
- Body font: 'Montserrat' (sans-serif) for a clean and readable user experience. Note: currently only Google Fonts are supported.
- Use minimalist, high-quality line icons to represent product categories and UI actions.
- Employ a clean, spacious layout with a focus on product imagery and readability.
